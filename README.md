# Chemi Code Website

## Common commands used

1) To deploy it to the live website use:

    ```
    npm deploy
    ```
2) To run the website locally use:

    ```
    npm start
    ```
3) Git commands:

    ```
    git add .
    git commit -m "commit name"
    git commit origin main
    ```
    
**Note: be sure to be in the repository directory**
